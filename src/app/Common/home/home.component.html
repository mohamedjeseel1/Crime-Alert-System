What is Crime Alert System?

By using this application, everyone can inform the crimes with the exact locations of that crime happens at a time without any ambiguous. Everyone can use this application from anywhere and anytime in safe and secure manner. It's like a bridge between the police and the people. Thus, everyone can stay updated and reduce crime in less amount of times.

How does it work?

There are three primary users. Admin, Police, and the General Public. Each user has the same plateform based on their user type. The Admin will act like a bridge between the police and general users. Everything was verified by the admin. They can be able to access several common features and some discrete features.

The system work flow is,

Common features: The admin can be able to create, retrieve, update, and delete the system data. Other users can view it.

Discrete features: Any individual can access the discrete features, which will be verified and resolved by the admin.
